# Analisi Rete Carburanti

Sistema di analisi e monitoraggio per reti di distributori carburante.

## Cosa fa

- Legge automaticamente i report XLS esportati da Fortech
- Aggrega i dati di tutte le stazioni della rete
- Calcola KPI mensili: litri erogati, fatturato, mix prodotto, mix pagamenti
- Rileva anomalie (cali rispetto al mese precedente e all'anno scorso)
- Genera report PDF mensile professionale
- Invia alert email automatici in caso di anomalie

## Stack tecnico

- Python 3.14
- Pandas / NumPy — analisi dati
- Matplotlib — grafici
- ReportLab — generazione PDF
- SQLite — storage storico
